Help to RTF for Linux

Purpose

Help to RTF is a file format converter converting Windows Help file (having the file extension .HLP), Multimedia Viewer Files (having the file extension .MVB) and Media View Library files (having the file extension .M14) into RTF files either.
  • To reproduce the source code as closely as possible OR
  • to produce an RTF-File useable as a printable handbook in good quality.

Linux-Version

This Linux-Version binary does not depend on a windows emulator and is free software. It is compiled for a i386 machine. It was compiled with gcc 2.7.2.3 running Kernel 2.0.36 on Suse Linux 6.0.

The linux version does not offer some features of the windows version related to the use of DLLs used in the help file. The Windows-Version of Help to RTF can be used with the windows emulator WINE.

Usage

A simple usage screen is displayed when starting the program without any parameters.

Warning: This program overwrites files already existing in the current directory having the same name as an already existing file. It is recommended to run the program in an empty directory.

Example:

mkdir myhelpfile
cd myhelpfile
hlp2rtflx -p -i -l -r ../myfile.hlp
